## 📖 Overview

This repository contains all laboratory exercises for the **Digital Systems Design Lab** course. The coursework focuses on designing, simulating, and implementing digital systems using **Verilog HDL** and **Intel Quartus Prime** software, targeting **FPGA** development.

All exercises were completed in collaboration with **[Pooria Rahmani](https://github.com/pooria1834)**.

## Exercises

| # | Topic |
|---|-------|
| 01 | Divisible by 3 & 11 circuits |
| 02 | Room entry control system |
| 03 | Combinational & sequential comparators |
| 04 | Stack implementation |
| 05 | Booth multiplier |
| 06 | Incubator controller |
| 07 | UART communication |
| 08 | Complex number ALU |
| 09 | Ternary Content Addressable Memory |
| 10 | Simple CPU |

## Tools

- Verilog HDL
- Intel Quartus Prime
- ModelSim
- DE2 FPGA Board

## Structure

Each exercise folder contains:
- Source code (`.v`, `.bdf`)
- Quartus project files (`.qpf`, `.qsf`)
- Lab reports (`.pdf`)
- Simulation results and images
